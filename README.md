# elspesialisten.no

Norges ledende leadplattform for elektriker og elektrotjenester.

## Tech stack

- **Next.js 15** – App Router, Server Components, ISR
- **TypeScript** – strikt typing
- **Tailwind CSS** – komplett design system (oransje primærfarge)
- **Framer Motion** – animasjoner (valgfri)
- **Google Fonts** – Syne + DM Sans + JetBrains Mono
- **Vercel** – hosting og edge functions

## Kjøre lokalt

```bash
npm install
npm run dev
# Åpne http://localhost:3000
```

## Bygge og deploye

```bash
npm run build
npm run start

# Deploy til Vercel
npx vercel --prod
```
